CS2 Analytics

Pipeline reproduzível de análise de partidas profissionais de Counter-Strike 2: bases consolidadas viram um banco SQLite, nove consultas SQL respondem as perguntas e este painel apresenta os resultados. O modelo preditivo permanece congelado — nada aqui o retreina.
